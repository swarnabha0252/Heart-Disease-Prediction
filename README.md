# Heart-Disease-Prediction
This project develops a machine learning classifier to predict the presence of heart disease based on clinical patient data. In a medical context, a **False Negative** (failing to identify a sick patient) report is significantly more dangerous than a **False Positive** report. Therefore, this project focuses on recall optimization along with the overall accuracy.

Dataset Source: https://www.kaggle.com/datasets/rishidamarla/heart-disease-prediction
For prediction, Logistic Regression model is used. Based on the dataset collected from Kaggle, 23% of Females have Heart Disease and 55% of Males have Heart Disease.
